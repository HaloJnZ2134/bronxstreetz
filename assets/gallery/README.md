# Gallery Images

Upload your Bronx Streetz screenshots in this folder.

The gallery page currently looks for these image files:

- `city.png`
- `cars.png`
- `cash.png`
- `map.png`
- `spawners.png`
- `community.png`

You can use `.jpg`, `.jpeg`, or `.webp` instead, but if you change the file extension, update the matching `src="assets/gallery/..."` path inside `gallery.html`.

Recommended image size: at least 1200px wide for clean cards on desktop.
