# Bronx Streetz Official Website

Static GitHub Pages website for the Roblox experience **Bronx Streetz**.

## Pages

- `index.html` - Home page
- `about.html` - Game information and rules
- `gallery.html` - Screenshot and promo gallery layout

## GitHub Pages setup

This site is ready to publish from the repository root.

1. Open the repository on GitHub.
2. Go to **Settings**.
3. Open **Pages**.
4. Set **Source** to **Deploy from a branch**.
5. Select branch **main** and folder **/(root)**.
6. Click **Save**.
